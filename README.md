## 📚 Courses & Repositories

Below is a list of the modules and subjects currently being taught. 

| **Course Name** | **Description** |
| :--- | :--- |
| **Systems and Networks** | Architecture, implementation, and management of networked systems and infrastructure. |
| **Computer Science** | Fundamental concepts of computation, algorithms, data structures, and computer architecture. |
| **AI** | Machine Learning algorithms, Neural Networks, and Natural Language Processing (NLP). |
| **Programming** | Software development lifecycle, design patterns, and high-level language implementation. |
| **Databases** | Relational model, schema optimization, indexing, and ACID compliance management. |
| **Networking** | OSI Model, routing protocols (OSPF, BGP), VLANs, and network security protocols. |
| **Web Development** | HTML, CSS, JavaScript, and Frontend Frameworks (React, Angular, or Vue). |
| **Programming** | Python, and Object-Oriented Programming (OOP) principles and abstraction. |
| **Networking** | Network fundamentals, TCP/IP, and Linux Administration. |
| **Cisco** | Routing and Switching, CCNA curriculum, and configuration via Cisco IOS. |
